# Sample Markdown Document

This is a **sample markdown** document to demonstrate the preview functionality.

## Features

The markdown preview supports:

- **Bold** and _italic_ text
- Lists (ordered and unordered)
- Code blocks
- Tables
- Links and images
- Blockquotes

## Code Example

```bash
#!/usr/bin/env bash

```
